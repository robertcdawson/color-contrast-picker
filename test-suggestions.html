Color Suggestion Test Page

Use this page to test the Chrome extension's color suggestion functionality.

Failing Color Combinations

These combinations should trigger suggestions when picked with the extension:

Yellow text on white background (#FFFF00 on #FFFFFF)
Should suggest darker text or darker background
Red text on light red background (#FF0000 on #FF8888)
Should suggest darker background or different text color
Gray text on similar gray background (#888888 on #999999)
Should suggest more contrasting colors

Passing Color Combination

Black text on white background (#000000 on #FFFFFF)
Should show "Colors meet WCAG standards"

Testing Instructions

  1. Open the Chrome extension
  2. Pick colors from the failing combinations above
  3. Verify that the Suggestions section auto-expands
  4. Check that both "Suggested Text Color" and "Suggested Background" appear
  5. Click "Apply" on any suggestion to test functionality
  6. Try the passing combination to verify it shows "Colors meet WCAG standards"
