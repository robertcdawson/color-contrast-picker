Color Contrast Picker Test Page

How to Test:

  1. Open the Color Contrast Picker extension
  2. Click on the foreground or background color squares
  3. Use the eyedropper to pick colors from the sections below
  4. Or manually type hex codes like #FF0000 in the input fields
  5. Check that the contrast analysis updates correctly
